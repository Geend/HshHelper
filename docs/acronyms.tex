\newdualentry{jwt}{JWT}{JSON Web Token}{Ein auf JSON basiertes Access-Token, standardisiert in RFC7519 \autocite{RFC7519}. Ermöglicht den Austausch von verifizierbaren Daten.}
\newdualentry{dfd}{DFD}{Datenflussdiagramm}{Ein Datenflussdiagramm zeigt über festgelegte Notation auf, welche Daten von welchem Akteur über welchen Eintrittspunkt bei welchem Prozess des Systems ankommen.}
\newdualentry{uuid}{UUID}{Universally Unique Identifier}{Ein Standard für Identifikatoren, der sich besonders dazu eignet, Informationen auf verteilten Systemen ohne zentrale Koordination eindeutig kennzeichnen zu können. \autocite{RFC4122}}
